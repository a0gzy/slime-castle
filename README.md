# Slime Castle Helper 🏰

Ультимативный помощник для игроков **Slime Castle**. Проект создан для систематизации игрового процесса и автоматизации рутинных расчетов.

## 🌟 Что уже есть в проекте

На данный момент реализованы основные инструменты, которыми пользуются топовые игроки и гильдии:

*   **Bingo Helper 🎲**: Умный алгоритм и тепловые карты для максимизации закрытых линий в ивенте Бинго.
*   **Monopoly Tracker 🎩**: Полный трекер для ивента Монополии. Отслеживает броски, автоматически передвигает фишку, считает очки и среднее значение за кубик. Имеет встроенную историю периодов.
*   **Rune Tracker 💎**: Инструмент для отслеживания вашей коллекции мифических рун и поиска дубликатов.
*   **Multi-language Support 🌍**: Полная поддержка русского и английского языков.

## 🚀 В планах развития

Проект постоянно дорабатывается. В ближайшем будущем ожидается:

1.  **Личный кабинет и Синхронизация ☁️**: Сохранение ваших данных в облаке для доступа с любого устройства.
2.  **Раздел Wiki 📚**: Детальные гайды по слаймам, боссам и артефактам.
3.  **Гильдейские инструменты 🤝**: Возможность делиться результатами и координировать действия внутри гильдии.

## 🛠 Техническая часть

Проект построен на современном стеке:
*   **Next.js 15** (App Router)
*   **Tailwind CSS**
*   **TypeScript**
*   **Lucide React** (Icons)

### Запуск локально

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) для просмотра.
